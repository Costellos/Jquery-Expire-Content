Welcome

Options

          
Below are all options available on the parent date_expire item :


          
          
          

            

              Option Code
              Status
              Default
              Description
            

            

              data-month
              Required
              null
              This defines the month you want the block to expire
            

            

              data-day
              Required
              null
              This defines the day you want the block to expire
            

            

              data-year
              Required
              null
              This defines the year you want the block to expire
            

            

              data-time
              Optional
              00:00:00:00
              This defines the time you want the block to expire
            

            

              data-hideparent
              Optional
              null
              If this is set, it will ignore any sub items, and hide the whole block after the date. can only be set to true. All other options will fail
            

          


          
          
You can have sub items that show or hide. This is useful to change only certain sub object, or to repalce whole sections on a website after a certain date.

          
Below are all options available on the sub item date_expire_action item:

          
          

            

              Option Code
              Action
              Description
            

            

              data-action
              hide_af
              This will hide eveything inside of this block after the parent date.
            

            

              data-action
              show_af
              This will show eveything inside of this block after the parent date.
