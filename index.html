Jquery-expire-content

A quick thing to create expire content blocks.

Welcome

I created this simple jquery to allow for quick show/hide of elemnts using javascript. It used data attributes to set dates that are used to either change, or hide the content on or after the date. See below for a demo.

Documentation

You will need to have jquery loaded to make this work.

Step 1

Create a element and give it the class "date_expire", also set the month, day, and year. You can also set the time, and the content, but these options are optional. When your done it should look like this:

```html
```

Step 2

Then inside of the element you just made, create 2 other elements. One with the class of "hide_af" and the other with a class of "show_af". The one with the class of "show_af" make sure you add "display:none;" to the style. It should look like this when your done.

```html
Content to hide after date.
Content to show after date.
```

Options

Below are all options available:

Option Code Status Default Description
data-month Required null This defines the month you want the block to expire
data-day Required null This defines the day you want the block to expire
data-year Required null This defines the year you want the block to expire
data-time Optional 00:00:00:00 This defines the time you want the block to expire
data-content Optional True This switches between hiding the whole block, or showing the content after. By default it shows the content after. Change it to false, and it will hide the whole block.
